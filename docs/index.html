Instalação

Menu Interativo

Abra o menu visual para selecionar skills:

npx workflow-skills

Auto-Update

Atualize skills já instaladas silenciosamente:

npx workflow-skills update

cURL (alternativa)

Execute o instalador via shell script:

curl -fsSL https://raw.githubusercontent.com/jpolvora/workflow-skills/main/install-skills.sh | bash

Contribuir

Skills vivem em .agents/skills/<nome>/SKILL.md com frontmatter YAML (name, description, version). Consulte AGENTS.md para o roteamento completo.
