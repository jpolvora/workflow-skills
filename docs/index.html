Workflows

Choose the right workflow orchestrator for your task scope. Both share shared/config.json and gates.md in dual-mode execution.

spec-to-pr

Standard FSM

Full multi-stage pipeline (Steps 0–9)

A comprehensive Finite State Machine orchestrator designed for end-to-end software lifecycle delivery from initial spec brainstorming down to PR shipping.

Pipeline Execution Steps:
  • Collaborative Spec Brainstorming (Step 0)
  • Architectural Review & Interview (Steps 1–2)
  • DAG Task Decomposition (Step 3)
  • Implement, verify & local code review (Steps 4–6)
  • Pre-PR testing, shipping & goal-fix-pr (Steps 7–9)

spec-to-pr-lite

Lite FSM

Fast sequential pipeline (Steps 0–5)

A streamlined alternative designed to run planning, sequential coding, code review, and shipping in a rapid direct feedback loop.

Pipeline Execution Steps:
  • Planning & brainstorm from existing spec
  • Direct sequential coding (no complex DAG required)
  • Quick local code review & automatic fix cycles
  • Fast consolidation & PR delivery
  • Zero setup/config duplication in dual-mode

Installation, Updates & Management

Canonical installation form: npx --yes github:jpolvora/workflow-skills. Preserves consumer-owned files under .agents/skills/shared/.

Interactive Menu

Launch the interactive visual installer to select skills and packages.

npx --yes github:jpolvora/workflow-skills

Update Skills

Refresh managed skills listed in shared/installed-skills.json while preserving local config.

npx --yes github:jpolvora/workflow-skills update
npx --yes github:jpolvora/workflow-skills update --include-new

Uninstall Skills

Remove specified skills with automatic dependency cascading.

npx --yes github:jpolvora/workflow-skills uninstall --skills goal-fix-pr --yes

Non-Interactive / CI

Automated unattended installation for CI/CD pipelines and agent harnesses.

npx --yes github:jpolvora/workflow-skills install --package workflows --yes

Version & Health Check

Verify current local installed version against latest remote release.

npx --yes github:jpolvora/workflow-skills --check
npx --yes github:jpolvora/workflow-skills --version

cURL Quick Shim

Execute CLI directly via shell pipeline (requires Node.js).

curl -fsSL https://raw.githubusercontent.com/jpolvora/workflow-skills/main/install-skills.sh | bash -s --
